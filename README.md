# Welfare Trust Website

This is a responsive website built for a welfare trust organization. The project 
is designed to provide information about the trust, its services, and how people can 
contribute or get help.

## Features
- Home page with introduction to the trust
- About section describing mission and vision
- Services/initiatives overview
- Contact form (basic front-end)
- Responsive design for mobile and desktop
- Interactive elements using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice front-end development skills and build a real-world
style informational website for a welfare organization.

## Note
This is a front-end only project. It does not include backend functionality such as form
submission storage or database integration.
